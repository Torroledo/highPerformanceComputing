# highPerformanceComputing

Basic implementations of High Performance Computing Based on 
the book of Gary W. Sabot, HIGH PERFORMANCE COMPUTING


Techniques:

	- Code Branching 
	- Data Branching 
	
	Methods of Iteration 
	
	- Point-Jacobi method
	- Gauss Seidel method
	- Red Black Gauss Seidel method
